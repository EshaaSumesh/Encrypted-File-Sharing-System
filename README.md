#  Encrypted File Sharing System

A secure file sharing system built using Python that implements **hybrid encryption**  
combining **AES symmetric encryption** with **RSA public-key cryptography**.

##  Features
- User authentication
- RSA key pair generation
- AES encryption for files
- Secure key exchange
- Encrypted file storage
- Authorized decryption and retrieval

##  Technologies
- Python
- Cryptography Library
- RSA, AES, Hybrid Encryption
- Git & GitHub

##  How to Run
```bash
pip install -r requirements.txt
python main.py
